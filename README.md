Ever wanted to have the practical use of desktop icons without the hideous blanket covering your well make backgrounds? Well, this is a skin for Rainmeter that creates a image that copies your background. This will cover your desktop icons with A fake background!

Steps:
1.download rmskin file and open.

2.set transparency to 100% and On Hover to Fade Out

3.Optional, set to bottom position to help keep it a little cleaner with other skins

4.edit the Mimic by right clicking and hitting edit. Change the "WWW" and "HHH" value to desired width and height!


If using a multi monitor set-up, set display monitor in the Manage menu to auto-select or manually select monitor.

THIS REQUIRES RAINMETER TO FUNCTION
